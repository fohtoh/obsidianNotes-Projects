# GEMINI.md

## Directory Overview

This directory is for studying the gospel of Jesus Christ from the Church of Jesus Christ of Latter-day Saints. It will be used to look through scriptures and other church references to help in gospel study.

## Key Files

- `README.md`: Explains the structure of the directory.
- `comeFollowMe/`: Contains materials for the Come Follow Me curriculum.

## Usage

The contents of this directory are intended to be used for personal gospel study. The `scriptures` directory will contain notes and thoughts on specific scriptures. The `topics` directory will contain notes and thoughts on specific gospel topics. The `comeFollowMe` directory will contain materials for the Come Follow Me curriculum.
